SciPyCon Argentina 2014

Bahía Blanca, Oct. 20-24.

Tuesday 5:15 p.m.–5:45 p.m.

Desarrollando mecánica de fluidos con Python

Agustin Ferrario

Audience level:
Novice

Description

La charla se basa en comparaciones de código de un algoritmo conocido com Random Flow Generator escrito en Python y el código original escrito en C y Fortran. Además de contar un poco la experiencia de un "no programador" acercandose a la computación científica por medio de python.

Abstract

La charla se basa en un conocido algoritmo utilizado en la mecánica de fluidos conocido como Random Flow Generator (RFG) desarrollado por A. Smirnov et. al. Actualmente el mismo algoritmo se encuentra en desarrollo por el Grupo de Investigación en Mecánica de Fluidos (GIMEF) de la Facultad Regional Resistencia de la UTN.

Durante la charla se mostrará comparacion la performance del código original contra una versión desarrollada en python. Así tambien las ventajas y desventajas del uso de python para la mecánica de fluidos (tiempos de desarrollo, disponibilidad de librerías, etc). Además como notas de color, se inscluirán algunos de los problemas de que un "no programador" desarrollando en Python.
